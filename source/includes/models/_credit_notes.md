POST https://www.octobat.com/api/credit_notes

# Example request
$ curl https://apiv2.octobat.com/credit_notes \

   -d customer="oc_cu_1459413729au6o6a9ae061" \
   -d description='Refund Subscription' \
   -d currency='EUR' \
